# Data Protection & Privacy

This folder contains statutes, rules, and regulations related to data protection and privacy in India.

## Current Law

### Digital Personal Data Protection Act, 2023 (DPDP Act)
- **Notification Date**: August 11, 2023
- **Status**: Enacted, rules pending
- **Key Provisions**:
  - Consent-based data processing
  - Rights of data principals
  - Obligations of data fiduciaries
  - Data Protection Board establishment
  - Cross-border data transfer provisions
  - Children's data protection

## Pending Rules

- DPDP Rules: Draft expected/released, final rules pending notification
- Check `drafts/` folder for latest versions

## Related Legislation

- IT Act, 2000 (Section 43A, Section 72A)
- IT (Reasonable Security Practices) Rules, 2011
- Consumer Protection Act, 2019
